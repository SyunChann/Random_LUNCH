# 🥢 서대문역 점심 메뉴 돌림판

서대문역 인근의 다양한 점심 메뉴를 랜덤으로 추천하는 웹 애플리케이션입니다.  
메뉴 고민을 덜어주고, 다양한 음식점을 탐방할 수 있도록 도와줍니다! 🚀

---

## 📌 기능 소개
✅ **랜덤 점심 메뉴 추천**  
✅ **음식 종류, 가격, 맛, 위치 등 정보 제공**  
✅ **네이버 지도 링크 제공 (길찾기 바로가기)**  
✅ **Google 스프레드시트 기반 데이터 관리**
